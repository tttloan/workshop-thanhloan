# Summary Report: "Cloud Mastery Series #2: DevOps Fundamentals & Infrastructure"

### Event Objectives

- **Systematizing IaC knowledge:** Comprehensive Infrastructure as Code and practical implementation with Terraform on AWS.
- **Mastering Container Orchestration:** Kubernetes (K8s) architecture and modern application management in Cloud-native environments.
- **Optimizing operations with Elixir:** Elixir/Erlang solutions for high-availability DevOps systems and fault tolerance.
- **Practical Hands-on:** Live demonstrations of infrastructure and application deployment techniques.

- **Thinh Nguyen** – FCAJ Cloud Engineer Trainee (IaC & Terraform Expert)

- Infrastructure as Code (IaC) with Terraform automation tools

- Cost Optimization: Technology selection for superior ROI
