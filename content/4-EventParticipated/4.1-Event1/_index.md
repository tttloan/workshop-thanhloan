# Summary Report: "Cloud Mastery Series #1: AI From Scratch"

![Cloud Mastery 2026 #1](event1-banner.png)

### Event Information

- **Event Name:** Cloud Mastery Series #1: AI From Scratch
- **Date & Time:** 09:00 – 12:00, March 14, 2026
- **Location:** 26th Floor, Bitexco Financial Tower, 02 Hai Trieu Street, District 1, Ho Chi Minh City
- **Role:** Attendee

### Event Objectives

- Explore practical applications of Generative AI in project development
- Learn advanced Prompt Engineering techniques
- Understand building self-operating AI Agents with function integration (Tool Use)
- Explore AIoT demo projects in real-life scenarios

### Key Highlights

**Advanced Prompt Engineering:** Chain-of-Thought, Self-Consistency, Tree-of-Thoughts, and RAG optimization techniques

**AI Agents with Strands:** Multi-step planning, tool integration, and autonomous decision-making

**Real-world AIoT Solution:** Locker Management System combining Raspberry Pi, Arduino, AWS Rekognition, S3, DynamoDB, and Lambda

### Key Takeaways

- Define clear roles and contexts for AI to receive higher quality results
- Agentic Architecture: Let AI use tools autonomously rather than provide simple text responses
- System Integration: Connect IoT hardware and AI Cloud services to solve operational problems
- Leverage tools like Amazon Q Developer and Proptimizer for workflow optimization

### Application to Work

- Optimize EduTrust project with Prompt Engineering for personalized learning recommendations
- Experiment with integrating AI Agents to automate user response processes
- Apply "Be Clear & Specific" principles when working with AI for coding assistance
